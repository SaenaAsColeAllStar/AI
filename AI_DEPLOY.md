## Manual Reference (Legacy)

The following steps are **automated by bootstrap**. Kept for troubleshooting.

### 1. Dependencies

```bash
sudo apt update
sudo apt install -y git curl wget build-essential python3 python3-pip nodejs npm
```

Or: `bash bootstrap/install-runtime.sh`

### 2. Ollama

```bash
curl -fsSL https://ollama.com/install.sh | sh
sudo systemctl start ollama
curl http://127.0.0.1:11434/api/tags
```

Or: `bash bootstrap/install-ollama.sh`

### 3. Qwen 32B

```bash
ollama pull qwen2.5-coder:32b
ollama list
```

Or: `bash bootstrap/install-model.sh`

### 4. OpenCode

```bash
npm install -g opencode-ai
```

Config template installed to `~/.config/opencode/opencode.jsonc` by bootstrap.

Or: `bash bootstrap/install-opencode.sh`

### 5. Verify Workstation

```bash
bash bootstrap/verify.sh
bash bootstrap/status.sh
```
